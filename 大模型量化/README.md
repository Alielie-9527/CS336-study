# 模型量化相关笔记
大模型量化主要就是PTQ（Post-Training Quantization）和QAT（Quantization Aware Training）两种方式，前者是直接对训练好的模型进行量化，后者是在训练过程中引入量化操作以提升量化后的模型性能。

当前进度：
- 已完成对PTQ的基本理解和实践，掌握了常见的量化方法如对称量化、非对称量化、动态量化等。
待完成：
- 继续深入学习QAT的原理和实现方法，了解其在大模型中的应用。
- 探索混合精度量化和自适应量化等高级技术。
